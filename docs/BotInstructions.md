## Project Overview

Telegram bot for recruiting employees.

## Telegram Bot core functionality

### General functionality

Bot has to be available in 3 languages.

For starters, bot enables user to choose language and then choose between 2 options: "Sprawdź oferty pracy" or "Skontaktuj się z nami".
